# Illuminknob Website

A modern, accessible marketing website for Illuminknob built with Next.js, React, Tailwind CSS, and TypeScript.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Useful Commands

```bash
npm run lint
npm run build
```
